Plugin Permissions

StarryDrive requires Roblox Studio plugin permissions to create and modify vehicle components during setup.

Enable Plugin Script Injection

If StarryDrive opens correctly but cannot apply changes, your Studio security settings may be blocking plugin scripts.

  1. Open Roblox Studio Settings.
  2. Navigate to the Security section.
  3. Enable plugin script permissions.
  4. Restart Roblox Studio.

Still Having Issues?

Try reinstalling the plugin and restarting Studio completely.

If the issue persists, contact support through Discord with:

  • Your Roblox Studio version.
  • Your operating system.
  • A screenshot of the error.
