Welcome to StarryDrive Documentation

StarryDrive is a Roblox Studio plugin designed to make creating vehicles simple, reliable, and customizable. Convert almost any vehicle model into a fully functional driving system without spending hours manually configuring constraints and physics.

Using the StarryDrive Setup panel, you can quickly configure a vehicle, apply presets, and fine-tune driving behavior such as suspension, acceleration, steering, speed, and more.

All settings are stored directly on the vehicle as Roblox Attributes, allowing you to easily adjust values later or reapply configurations whenever your vehicle changes.

Getting Started

If this is your first time using StarryDrive, start with the Quick Start Guide. It will walk you through creating your first working vehicle in just a few minutes.

For more advanced customization, see the Configuration and Attributes Reference pages.
