Create Your First Vehicle

Follow these steps to convert a model into a working StarryDrive vehicle.

1. Open the StarryDrive Setup Panel

Open Roblox Studio, navigate to the Plugins tab, and click the StarryDrive button.

If you cannot see the plugin, check the Installation and Permissions pages.

2. Select Your Vehicle Model

Select the model you want to convert into a vehicle, then click the vehicle selection button inside the StarryDrive panel.

3. Select the Primary Part

Choose the part that represents the base of your vehicle.

This part will act as the physical reference point for the vehicle, so it should be positioned level with the bottom of the vehicle or wheels.

  • Keep the height around 1 stud for consistency.
  • You can resize the part if necessary.
  • The part can be made invisible after setup.

4. Select the Vehicle Seat

Select the VehicleSeat used to control the vehicle.

The VehicleSeat must be directly inside the vehicle model. Do not place it inside another model or inside the PrimaryPart.

5. Choose a Preset

Select a preset such as Kart or Car to automatically configure your vehicle.

6. Apply and Test

Click Apply, then run your game to test the vehicle.

You can return to the setup panel at any time to apply new configurations or manually adjust settings.
