Installing StarryDrive

StarryDrive is distributed as a Roblox Studio plugin file. Follow these steps to install it manually.

1. Locate Your Plugin Folder

Open your Roblox Studio plugins folder.

The default location is: 

Windows:
%localappdata%\Roblox\Plugins

macOS:
~/Documents/Roblox/Plugins

2. Add the Plugin File

Move the StarryDrive plugin file into this folder.

3. Restart Roblox Studio

Close and reopen Roblox Studio. StarryDrive should now appear under the Plugins tab.

If the plugin does not appear, check the Permissions page.
