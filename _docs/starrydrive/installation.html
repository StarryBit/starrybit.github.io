Installing StarryDrive

StarryDrive is distributed as a Roblox Studio plugin file and is installed manually. Follow the steps below to get started.

1. Open the Plugins Folder

Open Roblox Studio's local plugins directory.

The default location is: 

Windows:
%localappdata%\Roblox\Plugins

macOS:
~/Documents/Roblox/Plugins

2. Install the Plugin

Move the downloaded .rbxm plugin file into the Plugins folder.

3. Restart Roblox Studio

Close and reopen Roblox Studio. StarryDrive should now appear in the Plugins tab.

4. Generate Your Scripts

The first time you use StarryDrive to generate scripts, Roblox Studio may display a permission dialog asking whether the plugin can insert scripts into your place.

  • Select Allow to grant the required permission.
  • After granting permission, click Apply again to generate the scripts.
  • This permission only needs to be granted once. Future generations will work without prompting again.
Note: If you deny this permission, StarryDrive cannot create scripts in your experience.

If the plugin doesn't appear in Studio or you're having permission issues, see the Permissions page.
