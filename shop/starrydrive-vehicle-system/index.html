Technical Breakdown

Spend less time tweaking constraints and balancing tedious forces, and more time elsewhere with the StarryDrive plugin. Use the StarryDrive Setup panel to easily configure an otherwise basic model into a vehicle, and either pick from the ready-made presets, such as a kart or car, or tune each of the settings, like suspension and speed, yourself. All settings are stored as attributes, making it easy to go back and adjust them as needed or use the panel to reapply new settings. There is thorough documentation and a quick-start guide to get you started. Customers are welcome to join the Discord server for support and updates.

Why I Made This

I found the hard way that relying solely on Roblox constraints for vehicles is a strenuous task that results in a vehicle ready to launch into the stratosphere the second I hit a weird bump in the ground. They can be great for a custom and realistic feel, but if you're like me and want something reliable, consistent, and easy to use, then if it weren't me, you likely would have made this plugin yourself. It's as I always say: time is money, and this plugin saves time.
